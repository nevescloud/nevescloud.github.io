mcp-rtc bridge

This page bridges the remote MCP server identified by the URL fragment (neves.cloud/b/#<site-id>) over WebRTC. To call its tools from a stdio MCP client, install the bridge once:

# Claude Code
claude mcp add bridge -- npx -y @nevescloud/mcp-rtc-bridge

# Codex CLI
codex mcp add bridge -- npx -y @nevescloud/mcp-rtc-bridge

# Cursor / Claude Desktop / any stdio MCP client — add to mcpServers config:
{ "mcpServers": { "bridge": { "command": "npx", "args": ["-y", "@nevescloud/mcp-rtc-bridge"] } } }

Then: connect({ siteId: "<id-after-#-in-this-URL>" }). The bridge speaks standard MCP — listTools, callTool.

For Claude.ai web with the Anthropic Chrome extension: open this URL in that tab; the bridge mounts automatically.
